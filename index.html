A* Pathfinding Demo

Click and drag on the tiles to draw obstacles.
Click and drag on the obstacles to erase them.

Click and drag the Start and Goal tiles to move them.

Click the Random button to randomly place obstacles.
Click the Run button to start pathfinding and the Clear button to clear the board.

This demo is best experienced on a desktop browser.
If you are using a desktop browser, try expanding the window to display desktop directions.
